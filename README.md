# pixi-experimental
Experimentation with pixi.js

## Getting started

### Start game locally
```
git clone https://github.com/fraxken/pixi-experimental.git
cd pixi-experimental
npm ci
npm start
```

### Start in dev mode

Open a first terminal:
```
git clone https://github.com/fraxken/pixi-experimental.git
cd pixi-experimental
npm ci
npm start
```

Then, open an other terminal:
```
npm watch
```
